\mbox{\hyperlink{classLogger}{Logger}} class for logging messages to console and file with different log levels. This class is implemented as a singleton to ensure a single instance throughout the application. \begin{DoxyNote}{Note}
\mbox{\hyperlink{classThread}{Thread}} safety is ensured for console output and file output 

You must call set\+Output\+File before logging to file, if you didn\textquotesingle{}t provide a filename a timestamped one will be generated
\end{DoxyNote}
