\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TType$>$\newline
class Pool$<$ TType $>$}

\mbox{\hyperlink{classPool}{Pool}} of reusable memory. 

RAII is used to manage the lifetime of objects.

The \mbox{\hyperlink{classPool}{Pool}} pre-\/allocates objects of type {\ttfamily TType} and reuses them to avoid frequent allocations and deallocations.

\begin{DoxyWarning}{Warning}
If you resize the pool, all existing objects are destroyed.
\end{DoxyWarning}
